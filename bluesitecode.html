About The Code

On this project, I assisted with some custom code along with input for site design, UX and implementation.

One of the original directions we had gone with this project was to develop a portfolio page that would display an image tile that would be in grey-scale, then on hover, it would fade into color. The image tile was a link to a page with more information on that featured company. By fading to a colored image, the link was intuitive to the user along with some directional copy to guide them.
