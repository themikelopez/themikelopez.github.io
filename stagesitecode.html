Creating a new logo design

Working with the lead designer, we created a modern logo to reflect their professional services.

Redesigning their website to help guide their clients.

Creating a more fluid flow of the Stage site, we were able to streamline the UX and help direct them where they wanted to be.

Asset creation and manipulation

I shadowed the lead designer while they created custom assets for the portfolio showcase. Adobe XD, Adobe Photoshop, and other useful design tools were used to achieve what the client was looking for at the time.

About The Work

On this project, I assisted with some custom code along with input for site design, UX and implementation.

One of the original directions we had gone with this project was to develop a portfolio page that would display an image tile that would be in grey-scale, then on hover, it would fade into color. The image tile was a link to a page with more information on that featured company. By fading to a colored image, the link was intuitive to the user along with some directional copy to guide them.
